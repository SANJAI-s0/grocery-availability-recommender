Grocery Availability Recommender

Production-oriented UI (React). If React fails to mount, use the links to test the backend directly.

Quick test links (backend)

Backend health — GET /health Available domains — GET /api/domains Products (grocery) — GET /api/products?domain=grocery Products (electronics) — GET /api/products?domain=electronics

Notes: Frontend will be served from http://localhost:3000 (nginx). If that URL is blank check Docker Desktop or the compose logs.

Status / Example

POST /api/predict-availability

{ "sales": 22, "day": 6 }
Tip: If frontend doesn't show, check that Docker mapped port 3000 to your host and that the React build completed successfully during `docker compose build`. To see frontend static files directly, open http://localhost:3000.
